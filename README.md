# html-sass
startup app for getting started with html-sass.

## Usage
- Clone repo

```
$ git clone git@github.com:pradeep1991singh/html-sass.git
```

- Change directory

```
$ cd html-sass
```

- Open terminal and type

```
$ npm install
```

- Run html-sass application

```
$ gulp serve
```

Or simply

```
$ gulp
```

- For production

```
$ gulp prod
```
